Hyrax: An open-source repository engine from Samvera

Learn more

Community Support

Hyrax is maintained and supported by Samvera, a global community of technologists who create and maintain repository software.

Flexible Workflow

Hyrax supports multiple workflows including the ability to define custom workflows, allowing implementations to tailor how Hyrax meets their deposit, review, and approval needs.

Customizable Solution

Hyrax offers a repository solution that can be customized for institutional/data repositories and digital object repositories.
